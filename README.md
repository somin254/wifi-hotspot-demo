# Wi-Fi Hotspot Demo (M-Pesa Enabled)

This project is a proof-of-concept paid Wi-Fi hotspot system designed for hostels.
Users connect to Wi-Fi, choose a time-based package, pay via M-Pesa, and receive
temporary internet access.

## Features
- Captive portal simulation
- Time-based access control
- M-Pesa STK Push integration (sandbox/real)
- Automatic access expiry
- Modular backend design

## Tech Stack
- Python 3
- Flask
- Safaricom Daraja API
- HTML/CSS (simple UI)
